netdata-virt
============
(c) 2017 Red Hat inc.

virt plugin for [netdata](https://github.com/firehol/netdata).

Let you monitor your libvirt-managed VMs.

GPLv2 licensed.

More to come.


Installation
------------

    
    $ cd netdata-virt
    $ go build -v .
    # cp netdata-virt /usr/libexec/netdata/plugins.d/virt.plugin
    

(yes, that's it)
